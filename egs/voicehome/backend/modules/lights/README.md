### Module: Lights

Lightning system - one onboard LED on ESP8266 for now.

##### Moves

- `turn_led_on`

  - "zapni vestavěnou ledku", "rozsviť ledku", "led on"

- `turn_led_off`

  - "zhasni vestavěnou ledku", "vypni ledku", "led off"

- `binking_led`

  - "zablikej vestavěnou ledkou", "zablikej ledkou", "led blink"
